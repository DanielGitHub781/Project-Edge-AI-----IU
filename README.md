# Age, Gender and Emotion Recognition App

This project is a mobile application that recognizes **age group**, **gender**, and **emotion** from facial images using machine learning models.

## Overview
- Developed with **TensorFlow** and **Keras** for training and **Android Studio** for deployment.  
- Uses **UTKFace** and **AffectNet** datasets for age, gender and emotion labels.  
- Trained models were converted to **TensorFlow Lite** for mobile inference.  
- The app runs entirely **on-device**, without internet connection.
